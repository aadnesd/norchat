## Overview
This package defines a Norway-first AI support agent platform that competes with Chatbase by prioritizing fastest time-to-setup, broad ingestion sources, multi-channel deployment, strong integrations, and GDPR-aligned security. The design specifies a retrieval-augmented agent runtime with actions, escalation, and analytics. The plan sequences delivery from ingestion and retrieval through channels, actions, security, and Playwright-based visual testing.
