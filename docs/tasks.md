# Task Breakdown

This document tracks the implementation tasks derived from the Chatbase competitor spec.

## Planned Tasks

1. Scaffold monorepo + tenancy model (API + web + worker + shared).
2. Ingestion MVP: files + website crawl with job queue and status tracking.
3. Knowledge base + retrieval service (chunking, embeddings, retrieval API).
4. Agent runtime + chat API with streaming responses.
5. Web widget + help page deployment with domain allowlist.
6. Admin console onboarding flow (create agent → add sources → deploy widget).
7. Actions framework and first integrations (CRM escalation, Slack notify, Stripe billing).
8. Multi-channel connectors (email, Slack, WhatsApp, Zendesk, Salesforce).
9. Analytics and observability dashboards.
10. GDPR controls (deletion, retention), RBAC, audit logging.
11. Playwright visual tests for onboarding, widget, help page.
